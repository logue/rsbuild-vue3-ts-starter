This template helps you start developing with Vue 3 and TypeScript on the Rsbuild stack. The template uses Vue 3 `<script setup>` SFCs. See [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) for details.

_Just copy your `src` from Vite!_

Includes [vue-router](https://router.vuejs.org/) and [Pinia](https://pinia.vuejs.org/).

Linting and formatting are driven by a unified `pnpm lint` pipeline using [Rslint](https://rslint.rs/), [Prettier](https://prettier.io/), and [Biome](https://biomejs.dev/).

## Commands

The commands are designed to stay simple and CI-friendly.

| Command     | Description                                            |
| ----------- | ------------------------------------------------------ |
| dev         | Start dev server.                                      |
| lint        | Run all lint/format steps (`lint:*`) in one command.   |
| test        | Run tests with Rstest.                                 |
| test:watch  | Run tests in watch mode with Rstest.                   |
| build       | Build for production.                                  |
| build:clean | Clear production build files.                          |
| preview     | Run the program generated by the production build.     |
| build-only  | Build for production without checking. For Deploy use. |
| type-check  | Check vue markup.                                      |
